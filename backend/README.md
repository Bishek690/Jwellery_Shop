# Jewelry Backend

## Requirements
- Node.js
- MySQL

## Setup

1. Install dependencies:
   ```sh
   pnpm install
   # or
   npm install
   ```

2. Configure your MySQL database in `src/data-source.ts` (default: user=root, password=password, db=jewelry_db).

3. Run the development server:
   ```sh
   pnpm dev
   # or
   npm run dev
   ```

The backend will start on http://localhost:4000
